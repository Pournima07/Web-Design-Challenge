Summary: Example Latitude vs Temperature




Visualizations

The purpose of this project is to see different weather changes as you get closer to the equator. At beginning we pulled weather data from weather map API on over 500 cities. After collecting the data we used matplotlib to plot various aspects of the weather vs latitude. We looked at four different factors like humidity, cloudiness, wind and temperature. This site provides source data and visualizations created as a part of the analysis, as well as explanation and descriptions of any trends and correlations between them.
